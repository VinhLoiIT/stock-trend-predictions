# Usage

Run in the terminal:

```
uvicorn api:app --host 0.0.0.0 --port 8000
```

The easiest way to test is to open a browser, e.g., Google Chrome, and enter:
```
http://0.0.0.0:8000/docs
```

Here you can try out some availables API
